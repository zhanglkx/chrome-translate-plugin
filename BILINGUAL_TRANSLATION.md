## 🔍 语言检测工作原理

### 检测算法

```javascript
function detectLanguage(text) {
  // 检查文本中的中文字符比例
  const chineseRegex = /[\u4e00-\u9fff\u3040-\u309f\u30a0-\u30ff]/g;
  const chineseMatches = text.match(chineseRegex) || [];
  
  // 如果中文字符超过 30%，判定为中文
  if (chineseMatches.length / text.length > 0.3) {
    return 'chinese';
  }
  return 'english';
}
```

### 检测范围

| 类型 | 检测范围 | 示例 |
|------|---------|------|
| **汉字** | \u4e00-\u9fff | 中文、汉字 |
| **平假名** | \u3040-\u309f | ひらがな |
| **片假名** | \u30a0-\u30ff | カタカナ |

## 🎬 翻译流程

### 选中英文文本

```
选中: "Hello world" (英文)
     ↓
检测语言: English (英文)
     ↓
查词典? ✓ (简单词)
     ↓
词典命中 (hello → 你好, world → 世界)
     ↓
返回: "你好 世界" (从词典)
响应时间: ~50ms (最快)
```

### 选中中文文本

```
选中: "你好世界" (中文)
     ↓
检测语言: Chinese (中文)
     ↓
查词典? ✓ (简单词)
     ↓
双向词典命中 (你好 → hello, 世界 → world)
     ↓
返回: "Hello world" (从词典)
响应时间: ~50ms (最快)
```

### 复杂内容

```
选中: "How are you today?" (英文短句)
或： "你好啊，最近怎么样？" (中文短句)
     ↓
检测语言: English 或 Chinese
     ↓
查词典? ✗ (超过3个单词)
     ↓
调用 DeepSeek API
     ↓
API 自动检测并翻译
     ↓
返回翻译结果
响应时间: 1-3s (取决于网络)
```

## 💡 故障排查

### 问题: 中文文本不翻译

**可能原因**:
- 词典中没有该中文词条
- 禁用了词典或 API Key 未设置

**解决**:
```
1. 如果是简单词，在 dictionary.js 中添加词条
2. 检查设置中是否启用了词典
3. 检查 API Key 是否设置
```

### 问题: 混合语言判定不正确

**示例**:
```
输入: "Hello 你好"
预期: 根据比例判定（中文占 50%，判定为中文）
```

**调整阈值**:
可以修改 `detectLanguage()` 中的 `0.3` 阈值：
```javascript
if (chineseMatches.length / text.length > 0.3) { // 改为 0.5 更严格
  return 'chinese';
}
```
