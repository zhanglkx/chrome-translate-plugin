
### 1. 指示符小点
- **外观**: 12px × 12px 的绿色圆点
- **位置**: 在选中文本的右下角
- **效果**: 悬停时放大并增加阴影
- **颜色**: 渐变绿色 (#4CAF50 → #45a049)

### 2. 翻译弹窗
- **样式**: 现代化设计，绿色边框
- **动画**: 平滑的滑入动画
- **按钮**: 复制翻译、朗读
- **关闭**: 点击 × 或 ESC 键

## 🔧 技术实现

### 核心逻辑变化

#### 之前的流程
```javascript
document.addEventListener('mouseup', async (e) => {
  const text = selection.toString().trim();
  
  // 立即发送翻译请求
  chrome.runtime.sendMessage({
    type: 'TRANSLATE_REQUEST',
    text: text,
    position: { x: e.clientX, y: e.clientY }
  }, (response) => {
    // 立即显示翻译
    showTranslation(text, response.translation, position);
  });
});
```

#### 现在的流程
```javascript
document.addEventListener('mouseup', async (e) => {
  const text = selection.toString().trim();
  
  // 保存信息，显示小点，等待用户点击
  pendingTranslation = { text, position: { x: e.clientX, y: e.clientY } };
  showIndicatorDot({ x: e.clientX, y: e.clientY });
  
  // 只有当用户点击小点时才调用翻译
  indicatorDot.addEventListener('click', () => {
    chrome.runtime.sendMessage({
      type: 'TRANSLATE_REQUEST',
      text: pendingTranslation.text,
      position: pendingTranslation.position
    }, (response) => {
      showTranslation(text, response.translation, position);
    });
  });
});
```

## 💡 设计理念

这次改动遵循以下原则：

1. **最小化侵入** - 小点代替弹窗，不打断用户
2. **用户主动** - 翻译由用户触发，不强制
3. **视觉清晰** - 绿色小点显眼但不过分
4. **交互流畅** - 单击即翻译，简单快速
5. **易于取消** - 点击其他地方直接隐藏
